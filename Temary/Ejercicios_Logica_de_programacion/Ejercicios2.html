2) Programa una función para contar el número de veces que se repite una palabra en un texto largo, pe. miFuncion("hola mundo adios mundo", "mundo") devolverá 2.

  • Run
  • Código JS
                
  let text = document.getElementById("text-2-2-run").value;
  let searchWord = document.getElementById("word-2-2-run").value;
                
  let regex = new RegExp(`${searchWord}`,"ig");
  let Search = [...text.matchAll(regex)];

  M.toast({html: `Del texto: "${text}"`});
  M.toast({html: `Se encontraron: ${Search.length} coincidencias de la palabra ${searchWord}`});
  console.log("Segundo Ej. - ",Search);
