1) Programa una función que invierta las palabras de una cadena de texto, pe. miFuncion("Hola Mundo") devolverá "odnuM aloH".

  • Run
  • Código JS
let text = document.getElementById("text-2-1-run").value; let splitedText = text.split(""); let reversedText = splitedText.reverse().join(""); M.toast({html: `La palabra "${text}" inversa es: ${reversedText}`}); console.log("Primer ej - ", reversedText);

2) Programa una función para contar el número de veces que se repite una palabra en un texto largo, pe. miFuncion("hola mundo adios mundo", "mundo") devolverá 2.

  • Run
  • Código JS
let text = document.getElementById("text-2-2-run").value; let searchWord = document.getElementById("word-2-2-run").value; let regex = new RegExp(`${searchWord}`,"ig"); let Search = [...text.matchAll(regex)]; M.toast({html: `Del texto: "${text}"`}); M.toast({html: `Se encontraron: ${Search.length} coincidencias de la palabra ${searchWord}`}); console.log("Segundo Ej. - ",Search);
