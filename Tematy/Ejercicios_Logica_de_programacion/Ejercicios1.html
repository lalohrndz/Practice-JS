Ejercicios de Lógica de programación 1 / 10

1) Programa una función que cuente el número de caracteres de una cadena de texto, pe. miFuncion("Hola Mundo") devolverá 10.

  • Run
  • Código JS
const nCaracteres = () => { let text = document.getElementById("text-1-1-run").value; M.toast({html: `La palabra "${text}" tiene ${text.length} caracteres`}); console.log("Primer ejercicio - ",text.length); }

2) Programa una función que te devuelva el texto recortado según el número de caracteres indicados, pe. miFuncion("Hola Mundo", 4) devolverá "Hola".

  • Run
  • Código JS
const nSliceCaracteres = () =>{ let text = document.getElementById("text-1-2-run").value; let number = document.getElementById("number-1-2-run").value; let slicedText = text.slice(0,number); M.toast({html:`De la frase "${text}", si recortamos ${number} quedaría así: ${slicedText}`}); console.log("Segundo ejercicio - ",slicedText); }

3) Programa una función que dada una String te devuelva un Array de textos separados por cierto caracter, pe. miFuncion('hola que tal', ' ') devolverá ['hola', 'que', 'tal'].

  • Código JS
const splitCaracteres = () =>{ let text = document.getElementById("text-1-3-run").value; let char = document.getElementById("char-1-3-run").value; let splitedText = text.split(char) M.toast({html:`Del texto: ${text} el array regresa: ${splitedText}`}) console.log("Tercer ejercicio - ",splitedText); }
